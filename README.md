# simple-uglify

> 用于对一些例外的静态文件进行压缩的一个小工具

## Install

``` bash
$ npm install
```

## Run

### Default task

minify all files

``` bash
$ gulp
```

### Img task

minify images

``` bash
$ gulp img
```

### Css task

minify css files

``` bash
$ gulp css
```

### Js task

minify js files

```bash
$ gulp js
```
